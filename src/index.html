Análisis de Datos

El análisis de los datos, se puede observar que hubo un incremento de muertes desde los años 2005 hasta 2008 que fue el top de asesinatos, el dataset no trae datos adicionales que puedan dar el porqué de este incremento; pero se puede evaluar la situación del país para estos años que coinciden con el inicio de la crisis económica Española, podría existir una correlación directa.
Justificación del gráfico escogido

El grafico escogido es un Bar Chart esto debido a que la información que se quiere presentar es una comparación en el tiempo por año mas exactamente, de la cantidad de mujeres asesinadas; para lo cual el grafico muestra el contrate de la variable tiempo (año) con la cantidad de muertes.
Fuente de Datos Escogida

La fuente de datos escogida es un dataset que relaciona datos de las mujeres asesinadas en Españas por sus compañeros o exparejas entre los años 2003 y 2017. La data viene de http://estadisticasviolenciagenero.msssi.gob.es/ página de estadísticas de Violencia de Genero. El dataset de nombre DomesticViolenceSpain.csv incluye datos como Year, Month, Autonomous Comunity, Province, Victim's age group, Patner / ex patner, Cohabitation, Existence o previous police report of gender violence, Number of victims.
